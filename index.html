Basic Intro. of HTML and WWW

How does World Wide Web work

The World Wide Web (www) is an information system of interlinked hypertext documents that are accessed via the Internet. It has also commonly become known simply as the Web.

What is HTML

HTML stands for Hypertext Markup Language, and it is how the content on the web is displayed on web browsers, such as, Google Chrome, Safari...In short, HTML provides a framework for webpages.

Components of HTML

Elements: All HTML documents are made of HTML elements and each element is enclosed in angle brackets or Tags (< >).

Tag: An HTML tag is always contained within angled brackets (< >). Though most tags have an opening and closing tag, some tages - Void Tags - do not require a closing tag.

Attribute: An Attribute is a property of an HTML element. For instance, to add a link (Google) to words "I'm feeling lucky", we set href attribute of an anchor tag in this way.
I'm feeling lucky.
